多媒體應用

  1. 智慧聯網電視(Smart TV)相關技術,包括人臉識別、手勢辨識、語音識別、個人化服務、互動式視覺人機操作介面等。
  2. Super Remoter (HTML5-based跨平台Smart TV遙控器,可使用Android或iOS智慧手機做為Smart TV遙控器)
  3. H.264/H.265視訊編解碼技術
  4. 可調式視訊編碼(scalable video coding)技術
  5. 多視角影像深度估測技術 (shyang)
  6. 快速人臉識別(face recognition)技術
  7. 影像與視訊的錯誤隱藏(error concealment)技術
  8. Real-time MP3 encoder source code: optimized for TI C5X DSP and ARM CPU.
  9. Real-time MP3 decoder source code: optimized for TI C5X DSP and ARM CPU.
  10. MPEG-1/2 Layer II decoder source code for DVB-T: optimized in C-code level.
  11. AC3 decoder source code in C language
  12. DVD subpicture decoder source code in C language
  13. DVD Navigation control source code in C++ language
  14. Music identification technique (& source code) based on MPEG-7 audio signature. The technique can be applied to music database with content retrieval
  15. Digital image stabilization (視訊防手震技術)
  16. Wavelet toolbox for image/video applications
  17. Moving object detection from a moving-camera video sequence
  18. Android平台之行動數位電視DVB-H接收軟體，包含native service與使用者介面
  19. 視訊(H.264)與音訊(AAC)之3GP封裝軟體
  20. FLUTE/ESG parser軟體
  21. HTML 5 多媒體互動技術
  22. 地面投影體感互動技術與系統

網路應用

  1. Internet lawful interception – FTP& HTTP封包捕捉、協定分析、資料儲存與檔案及網頁之還原、SIP&H.323 VOIP即時監聽與錄音存證
  2. MPEG-DASH自適性串流系統技術
  3. 視訊編碼在異質覆疊網路多源應用層群播系統之優化
  4. GPRS/SMS/GPS-based 定位與基地台信號強度監測系統
  5. SIP-based VoIP 影像電話系統：含SIP extension functions, Security mechanism, Recording, and automatic bandwidth/encoding adjustment
  6. WiMAX MAC physical layer implementation
  7. 無線通訊系統應用
  8. 數位電視(DVB-T)接收機之解調變及通道解碼參考軟體
  9. 數位電視(DVB-T)接收機之TS (Transport Stream) parser軟體(source code)，包含「電子節目單」之解碼
  10. 行動數位電視(DVB-H)接收機之TS (Transport Stream) parser軟體(C++ source code)
  11. 行動數位電視(T-DMB)接收機之TS (Transport Stream) parser軟體(source code)，包含MPEG-4 sync layer及BIFS decoder
  12. 第二代數位電視(DVB-T2)接收機之解調變及通道解碼參考軟體
  13. 數位廣播(DAB)接收機之解調變及解碼參考軟體(source code)，包含audio decoder & PAD parser
  14. AMR編解碼參考軟體(source code，最佳化後之浮點數運算版本，比3GPP之參考軟體快很多)
  15. RFID室內定位及追蹤系統
  16. 軟體智慧型封包分類系統
  17. 支援可適性視訊編碼之內容傳遞網路系統設計
  18. 聯網電視系統之網路影音預錄平台技術
  19. 被動式光學網路系統之省電機制
